Mi Proyecto Personal

Esta página fue creada como parte de un proyecto personal para practicar HTML y CSS. El objetivo principal es aprender a estructurar una página web desde cero y aplicar estilos básicos.

Características del proyecto

  • Uso de etiquetas básicas de HTML
  • Aplicación de estilos con CSS externo
  • Diseño simple y fácil de entender
