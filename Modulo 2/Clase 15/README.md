\_\_# Módulo 2: Aprendizaje Esperado N° 6

## Actividad: Tienda Virtual

### Objetivo

AImplementar JQuery: Manipular DOM, gestionar eventos, usar plugin Boostrap-Query, crear página interactiva.

### Descripción

Eres parte del equipo encargado de diseñar una tienda vistual para una pequeña empresa. La página inicial debe mostrar una lista de productos con la capacidad de: Seleccionar productos favoritos, interacción dinámica e incorporar un carrisel de boostrap.

### Instrucciones

#### POO y Clases

-Configurar el entorno: Crea un archivo HTML y un archivo script e incluye la biblioteca JQuery en tu proyecto, junto con Bootstrap y sus dependencias.

-Estructura HTML:
\*Crear una lista de productos: incluye al menos 6 productos con una imagen, un nombrey un botón para marcarlos como favoritos.

\*Añadir un carrusel de Boostrap: muestra tres imágenes destacadas de los productos en un carrusel con transiciones automáticas utilizando el plugin Bootstrap.

-Interacciones con JQuery:
\*Usa JQuery para detectar cuando un usuario hace clic en el botón de "Favoritos" y añade o elimina el producto de la lista de favoritos.

\*Muestra un mensaje dinamico indicando si un producto fue añadido o eliminado.

-Manipulación del DOM:
\*Usa selectores JQuery para resaltar productos marcados como favoritos(Cambiar color o agregar un ícono).
